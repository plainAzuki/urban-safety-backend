# 参考文献・引用候補

この文書は卒業論文で参照候補となる資料を整理したものである。実際の提出時には、大学指定の引用形式に合わせて、閲覧日、発行年、URLを再確認する。

## 公的防災・気象情報

- 気象庁「気象庁防災情報XMLフォーマット」  
  URL: https://xml.kishou.go.jp/tec_material.html  
  本研究の気象データ取得・構造化の背景資料として利用できる。

- 気象庁「防災情報」  
  URL: https://www.jma.go.jp/bosai/  
  気象警報・注意報などの公的情報源として参照できる。

- 国土交通省「防災情報提供センター」  
  URL: https://www.mlit.go.jp/saigai/bosaijoho/  
  複数の防災情報を集約して提供する公的事例として参照できる。

- 内閣府「避難情報の判断・伝達」  
  URL: https://www.bousai.go.jp/oukyu/hinankankoku/index.html  
  避難情報の意味や伝達に関する背景資料として利用できる。

- 内閣府「避難情報に関するガイドライン」  
  URL: https://www.bousai.go.jp/oukyu/hinanjouhou/r3_hinanjouhou_guideline/index.html  
  避難情報の制度や警戒レベルに関する説明に利用できる。

## 本研究での扱い

本研究では、上記の公的情報源を参考にしつつ、実際の災害判断や避難判断を自動化することは目的としない。システムの評価では、公的情報と模擬データの区別、出典表示、更新時刻表示、統一データモデルによる表示可能性を中心に扱う。
